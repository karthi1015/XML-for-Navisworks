IFC naar Navisworks XML


Bedoeling:
Deze applicatie is bedoeld om geautomatiseerd
searchsets te maken voor Navisworks met behulp
van een IFC model

Werking:
Alle data in IFC wordt gezocht op type naamgeving
Vervolgens wordt er met de type naamgeving een searchset
gemaakt gecategoriseerd per IFC entiteit





Let op: Op dit moment kunnen alleen
IFC modellen gemaakt met Autodesk Revit, Archicad en Tekla
in deze tool worden geladen
.
